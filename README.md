# Queen's Student Page Sample

A sample repository for **CISC 121's Week 2 Homework** - launching your own Queen's School of Computing student webpage using GitHub Pages and completing peer code review.

## Overview

This repository serves as a reference and template for creating a personal Queen's Computing student webpage. Students will learn to build and deploy a live website while practicing collaborative code review skills.

## Tutorial

Follow the complete step-by-step tutorial at: **[https://ruslankain.github.io/queens-student-page/](https://ruslankain.github.io/queens-student-page/)**

The tutorial is designed for students with **no coding experience required** and can be completed in approximately **20 minutes**.

## What You'll Build

- A personal student webpage with Queen's School of Computing branding
- HTML structure with sections for About Me, Projects, and Contact
- CSS styling using Queen's official colors (red, gold, blue)
- Live deployment via GitHub Pages
- Experience with Git version control and collaborative code review

## Prerequisites

- GitHub account (free)
- Email verification enabled for GitHub Pages
- A partner for peer review (assigned in class)

## Deliverables for OnQ Submission

Once you complete all tutorial steps, submit the following to OnQ:

1. **Live URL**: Link to your deployed website
2. **Repository link**: Link to your GitHub repository with code  
3. **Peer review evidence**: Direct URL of your review on partner's repo (either Pull Request or Issue link)
4. **Screenshots**: 2 images showing your website and your peer review comments
5. **Summary**: 3 sentences covering biggest win, one quick fix, and one nice-to-have improvement

## Tutorial Steps Overview

1. **Step 0**: Sign in/create GitHub account (~1-2 min)
2. **Step 1**: Create your repository (~2 min)
3. **Step 2**: Add homepage HTML file (~4 min)
4. **Step 3**: Add Queen's branding CSS (~3 min)
5. **Step 4**: Enable GitHub Pages deployment (~3 min)
6. **Step 5**: Personalize and publish updates (~6-7 min)
7. **Step 6**: Partner peer review process (~20-25 min total)

## Key Learning Outcomes

- Understanding of basic HTML structure and semantic elements
- CSS styling and use of CSS custom properties (variables)
- Git workflow: commits, branches, and pull requests
- Collaborative development through code review
- Web deployment with GitHub Pages
- Professional communication in code review contexts
